 
Задача №5 Bash скриптинг
Description
Вывести загрузку CPU на страничку NGINX в реалтайме в %, страница должна обновляться автоматически. Создать демон процесс, который будет писать раз в 5 секунд логи NGINX в файл 1 (при попытке грохнуть процесс, он должен перезапускаться).
Когда файл 1 достигнет размера больше чем 300 кб, он должен очищаться.
В файле 2 должен присутствовать лог об удачных очистках файла 1 с датой и временем очистки, а также количеством удаленных записей.
В 3 файл складывать логи с 5хх.
В 4 файл складывать логи с 4хх.
Логи NGNIX можно сохранять любые до каких получится дотянуться. Для парсинга логов использовать sed и awk
Days: 2
