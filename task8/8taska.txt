Задача №7 AWS RDS
Description
Создать инстанс для базы данных типа postgres. Подключиться к ней через pgAdmin, установленный на ПК и через командную строку терминала linux (через терминальный клиент psql). Cоздать postgres базу данных jundb. Создать в ней таблицу под названием credit_cards_numbers. Руками сделать snapshot. Удалить базу данных. Восстановить её из snapshots. Проверить наличие созданной таблицы через pgAdmin и терминальный клиент psql. Настроить её бэкапы через AWS Backup по расписанию (раз в час например). Когда сделан автоматический backup, удалить текущий инстанс и восстановить его из бэкапа. Проверить наличие созданной таблицы credit_cards_numbers через pgAdmin и терминальный клиент psql.
Days: 2
